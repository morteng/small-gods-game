# Emoji Reference

Visual representation system for the world.

---

## Design Principle

Emojis are the visual language of the game. They must be:
- **Readable by LLMs** (they can "see" and reason about them)
- **Intuitive for humans** (immediate recognition)
- **Extensible** (new entities get appropriate emojis)

The LLM can suggest emojis for new entities it creates.
