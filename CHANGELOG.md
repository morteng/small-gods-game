# Changelog

All notable changes to Small Gods are recorded here. The file is generated from
[Conventional Commits](https://www.conventionalcommits.org) by
[commit-and-tag-version](https://github.com/absolute-version/commit-and-tag-version) —
run `npm run release` (see [docs/RELEASING.md](docs/RELEASING.md)) to cut a version.
Versioning follows [SemVer](https://semver.org); the project is pre-1.0 (`0.y.z`),
so minor bumps may carry breaking changes.
